# RealSourcing 高保真原型设计分析

## 设计风格总结

### 核心视觉特征
- **深色主题**：深紫色到黑色渐变背景 (#1a0b2e → #0a0a0a)
- **主色调**：紫色系 (#8b5cf6, #a855f7) - 科技感、AI 驱动
- **辅助色**：蓝色 (#3b82f6)、橙色 (#f97316)、绿色 (#10b981)
- **玻璃态效果**：卡片使用半透明背景 + 紫色边框光晕
- **字体**：Inter 20px，中英文混排

## 设计系统要点

### 颜色系统
```
Primary Purple: #8b5cf6, #a855f7
Secondary Blue: #3b82f6
Accent Orange: #f97316
Success Green: #10b981
Background: #0a0a0a → #1a0b2e (gradient)
Card Background: rgba(15, 15, 25, 0.6) with purple glow
Border: rgba(139, 92, 246, 0.3)
Text Primary: #ffffff
Text Secondary: rgba(255, 255, 255, 0.7)
```

### 组件特征
1. **卡片**：深色半透明背景 + 紫色发光边框 + 圆角
2. **按钮**：
   - Primary: 紫色渐变 (#8b5cf6 → #a855f7)
   - Secondary: 深色边框 + 半透明背景
   - CTA: 橙色 (#f97316)
3. **输入框**：深色背景 + 紫色边框 + 图标前缀
4. **状态标签**：
   - LIVE: 红色背景
   - UPCOMING: 蓝色背景
   - PAST: 灰色背景
5. **侧边栏**：深色背景 (#0f0f19) + 激活项紫色高亮
6. **数据卡片**：图标 + 数字 + 标签 + 趋势指示器

### 布局特点
- 侧边栏宽度：240px
- 主内容区：max-width 1440px
- 卡片间距：16-24px
- 圆角：8-12px
- 阴影：紫色光晕效果

## 执行摘要

本阶段完成了 RealSourcing 3.1 平台 Factory 页面的核心范式重构。这不是一次 UI 美化，而是一次**商业逻辑的根本性转变**：从"供给侧驱动"（阿里巴巴模式）转向"需求侧驱动"（社区口碑模式）。

**核心判断**：在 AI 时代，采购商真正需要的不是更多数据，而是**更少但更精准的决策信号**。"这家工厂懂我的市场吗？他们能支持我的销售模式吗？圈内人怎么说他们？"——这三个问题的答案，比任何传统指标都更能驱动采购决策。

**本阶段已交付**：AMR v2.0 工厂评估体系（四维雷达图）、渠道能力看板、买家体感词云、履约时效图、全球买家生态地图，全部落地至前端代码并推送至 GitHub。

### 1.2 我们删除了什么（同样重要）

> **"你添加了什么"不如"你删除了什么"更能体现产品判断力。**

以下内容被**有意识地移除**，每一项删除都有明确的商业理由：

| 删除的内容 | 删除理由 |
| :--- | :--- |
| **成立年限** | 资历不等于能力。一家成立 20 年但从未接触 FBA 的工厂，对电商卖家毫无价值 |
| **员工人数** | 采购商不关心工厂有多少人，他们关心能不能接小单、能不能快速打样 |
| **星级评分（1-5 星）** | 与阿里巴巴无差异，且容易被刷分。替换为基于交易数据的 AMR 综合指数 |
| **"AI 推荐理由"按钮** | 用户不需要点击才能看到推荐理由，关键信息应直接呈现在首屏 |
| **收藏/视频通话按钮组** | 卡片层级不是操作层级，减少干扰，聚焦信息传递 |

### 1.3 当前的局限性（诚实评估）

目前 AMR 数据是**基于 `overallScore` 推算的占位数据**，不是真实的 AI 计算结果。这意味着：

- 两家评分相近的工厂，AMR 四维分布几乎相同，缺乏区分度
- 买家体感词云是规则生成的，不是真实买家沉淀的口碑
- 渠道能力标签是基于工厂类别推断的，不是经过验证的实际能力

**这是当前最大的产品债务，必须在下一阶段优先偿还。**

## 二、核心增长假设

在规划下一步之前，我们需要明确驱动 RealSourcing 增长的核心假设：

> **假设一**：采购商在 Factory 页面的核心行为不是"浏览"，而是"筛选"。他们在 30 秒内决定是否值得进一步了解一家工厂。因此，**首屏信息密度和信号精准度**是转化率的最大杠杆。

> **假设二**：工厂的"渠道能力"（能否支持 Dropshipping、FBA、小单等）是采购商筛选的第一道过滤器，优先级高于价格和评分。

> **假设三**：**"圈内口碑"**（买家体感词云）比平台自评更可信，更能驱动采购决策。这是 RealSourcing 建立护城河的关键数据资产。

---

## 三、下一步计划：三个优先级

### 优先级 P0 — 数据真实化（2-4 周）

**这是最紧迫的工作，没有真实数据，AMR 模型只是一个漂亮的空壳。**

#### P0.1 工厂渠道能力数据采集

在工厂入驻流程中增加**结构化能力声明**，要求工厂勾选其支持的渠道能力：

```
□ Dropshipping（一件代发）
□ Amazon FBA（备货 + 贴标）
□ Shopify / 独立站
□ 展会参展（Canton Fair / CES 等）
□ 小单友好（MOQ < 50 件）
□ 白标 / OEM 发货
```

这些数据由工厂自申报，但需要经过**买家验证**才能显示"已验证"标记。

**商业价值**：渠道能力标签是采购商筛选的第一道过滤器，直接影响工厂的曝光量和询价转化率。工厂有强烈动机如实填写。

#### P0.2 买家体感标签系统

在每次 Meeting 结束后，向买家推送一个**5 秒微调研**：

```
这次会议中，您对这家工厂的感受是？（最多选 3 个）

[ 英文沟通流畅 ]  [ 快速打样 ]  [ 小单可接 ]
[ FBA 条码无缝 ]  [ 包装质感好 ]  [ 决策层直接沟通 ]
[ TikTok 爆款感知强 ]  [ 响应及时 ]  [ 价格透明 ]
```

这些标签**不是评分**，而是**体感描述**。它们会随着时间积累，形成工厂的"圈内口碑"，成为平台最有价值的数据资产之一。

**商业价值**：这是 RealSourcing 独有的数据，任何竞争对手无法复制。

#### P0.3 AMR 评分算法接入后端

将 AMR 四维评分从前端推算改为后端真实计算，数据来源：

| AMR 维度 | 数据来源 |
| :--- | :--- |
| **市场敏锐度** | 社媒活跃度 API + 展会参与记录 + 官网内容质量评分 |
| **渠道多元化** | 工厂自申报 + 买家验证标签 |
| **信息敏捷度** | 平台内首次响应时间 + 样品转化率 + Meeting 完成率 |
| **全球化广度** | 出口市场分布 + 国际认证数量 + 全球买家活跃数 |

---

### 优先级 P1 — 转化漏斗优化（4-8 周）

**目标**：将 Factory 页面的"浏览 → 询价"转化率提升 30%。

#### P1.1 智能匹配首屏

当前 Factory 列表是静态排序的。应改为**基于采购商画像的动态匹配**：

- 采购商首次登录时，完成一个 2 分钟的"采购需求画像"问卷（产品类别、目标市场、MOQ 要求、渠道模式）
- 系统根据画像，将最匹配的工厂排在前面，并在卡片上显示**"为您匹配"**标记和具体匹配原因

**商业价值**：个性化推荐是 Amazon 和 Netflix 的核心增长引擎，同样适用于 B2B 采购场景。

#### P1.2 "快速验证"机制

在工厂卡片上增加一个**"30 分钟快速视频验证"**按钮。采购商可以预约一个 30 分钟的快速视频通话，工厂展示产线、样品和关键能力。

这个功能直接打通了 RealSourcing 的核心差异化优势（实时视频 + AI 翻译），并将其嵌入到采购商的自然决策流程中。

**商业价值**：这是 RealSourcing 的"Aha Moment"——采购商第一次体验到"30 分钟内就能看到工厂产线"的震撼，是最强的留存和口碑驱动因素。

#### P1.3 工厂详情页的"社会证明"强化

当前详情页的评价模块过于简单（星级 + 文字）。应升级为：

- **买家身份验证**：显示评价者的平台交易记录（如"已完成 3 笔订单"），增加可信度
- **评价维度细化**：将评价拆分为"沟通效率"、"样品质量"、"交期准确"、"包装水平"四个维度
- **视频评价**：鼓励买家上传 30 秒的视频评价，展示收到的产品

---

### 优先级 P2 — 生态建设（8-16 周）

**目标**：建立 RealSourcing 独有的"采购商社区"护城河。

#### P2.1 导师认证体系

招募 50 名在电商领域有丰富采购经验的"采购导师"，赋予他们对工厂进行**人工背书**的权力。导师背书将以"⭐ 导师严选"金标的形式显示在工厂卡片上，并附上导师的具体评语。

**商业价值**：人工背书是最高可信度的信任信号，也是最难被竞争对手复制的护城河。

#### P2.2 "采购圈"社区功能

在平台内建立一个轻量级的采购商社区，允许买家：
- 分享采购心得和工厂评价
- 发起"联合采购"（多个小买家合并订单，满足工厂 MOQ）
- 提问和回答关于特定工厂或产品类别的问题

**商业价值**：社区是最强的用户留存工具，也是"买家体感词云"数据的主要来源。

#### P2.3 工厂 AMR 成长路径

向工厂展示其 AMR 评分的提升路径：

```
您的 AMR 评分：64 / 100

📈 提升建议：
- 完善渠道能力声明（+8 分）
- 获得 5 个买家体感标签验证（+6 分）
- 参加 1 次平台 Webinar（+4 分）
- 添加 Canton Fair 参展记录（+5 分）
```

**商业价值**：AMR 成长路径将工厂转化为平台的**主动参与者**，他们会自发地完善信息、参与活动，从而提升整个平台的数据质量。

## 四、关键指标体系

作为 GTM 首席增长官，我们需要追踪以下指标来验证上述假设：

### 4.1 采购商侧指标

| 指标 | 当前基线 | 90 天目标 | 衡量的假设 |
| :--- | :--- | :--- | :--- |
| **Factory 页面 → 询价转化率** | 待测量 | +30% | 精准信息展示降低决策摩擦 |
| **工厂卡片平均停留时间** | 待测量 | > 45 秒 | AMR 信息足够吸引深度阅读 |
| **"快速验证"预约率** | 0（功能未上线） | 15% 的详情页访问者 | 视频验证是核心转化杠杆 |
| **买家体感标签提交率** | 0（功能未上线） | > 60% 的 Meeting 后 | 微调研足够简单，买家愿意参与 |

## 五、风险与应对

| 风险 | 概率 | 影响 | 应对策略 |
| :--- | :--- | :--- | :--- |
| **工厂虚报渠道能力** | 高 | 高 | 买家验证机制 + 投诉降分系统 |
| **买家不愿填写体感标签** | 中 | 高 | 极简化设计（5 秒完成）+ 积分激励 |
| **AMR 算法被工厂"刷分"** | 中 | 高 | 多数据源交叉验证 + 人工审核高分工厂 |
| **新卡片设计降低点击率** | 低 | 中 | A/B 测试，保留旧卡片作为对照组 |

## 六、一句话战略定位

> RealSourcing 不是一个更好的阿里巴巴，而是一个**完全不同的东西**：它是全球采购商社区的**共识引擎**——在这里，工厂的价值由市场决定，而不是由平台年限决定。

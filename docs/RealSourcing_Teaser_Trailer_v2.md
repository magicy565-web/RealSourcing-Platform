# RealSourcing 概念片 v2.0：制作指导与优化脚本

**致：** RealSourcing 创始人

**发件人：** Manus AI (首席增长官视角)

**主题：** 从“产品介绍”到“品类宣言”——迪拜大会 75 秒必杀视频脚本 v2.0

---

Gemini 提供的 v1.0 脚本非常出色，它准确地将 FGPI 的四大支柱转化为了视觉语言。这是一个完美的“产品功能介绍片”。

但我们的目标是在迪拜的私密晚宴上，在 75 秒内，让台下那些身经百战的顶级 DTC 品牌创始人和亚马逊大卖，产生“**我过去十年赖以生存的模式，可能已经过时了**”的认知冲击。我们需要的不是一个“更好”的视频，而是一个能**定义品类、划分时代**的宣言。

这份 v2.0 方案，在 v1.0 的基础上，做了三个核心升级：

1.  **叙事升级：从“解决痛点”到“定义敌人”**
    -   v1.0 的敌人是“盲猜游戏”。v2.0 的敌人是“**增长税 (The Growth Tax)**”——即所有卖家在旧模式下，为信息不对称、沟通损耗和低效履约所付出的、看不见的、但持续流血的成本（时间、资金、机会）。这个词更能激发顶级卖家的共鸣与憎恨。

2.  **价值升级：从“增长伙伴”到“增长即服务 (Growth-as-a-Service)”**
    -   v1.0 的解决方案是“首席供应链增长伙伴”。v2.0 将其升维为“**增长即服务**”——一个更具 SaaS 时代感、更性感、更具颠覆性的概念。你购买的不是工厂资源，而是“持续增长”这个结果本身。

3.  **证据升级：从“悬浮 UI”到“真实产品 UI”**
    -   v1.0 的数据展示是后期特效。v2.0 将直接使用我们已经设计好的 **RealSourcing AMR 仪表盘和履约时效图的 UI 元素**，让视频中的每一个数据都感觉像是从一个真实、成熟、正在运行的系统中实时调取，极大增强可信度。

这份脚本可以直接交给顶级的广告制作公司，它包含了足够清晰的视觉、节奏和情绪指导。

## 导演简报 (Director's Brief)

以下是交给制作公司时，需要明确传达的核心创作方向。

### 视觉风格参考

这支视频的视觉基调应介于以下三者之间：

- **Apple iPhone 发布会宣传片**（极简、克制、每一帧都有力量）
- **Stripe 的品牌视频**（科技感、数据可视化、代码美学）
- **顶级对冲基金的机构宣传片**（权威、精准、不需要取悦所有人）

**坚决避免**：阿里巴巴式的"工厂航拍 + 员工整齐排队"，以及国内常见的"激情配乐 + 快切 + 大字幕"风格。

### 颜色系统

| 用途 | 颜色 | 说明 |
| :--- | :--- | :--- |
| 主背景 | `#0a0a0f` 近黑 | 代表旧世界的黑暗与未知 |
| 核心品牌色 | `#a78bfa` 幽紫 | RealSourcing 的 AI 智能感 |
| 增长/正向 | `#34d399` 电子绿 | 数据增长、订单确认 |
| 危险/旧世界 | `#f87171` 警告红 | 增长税、风险、冻结 |
| 数据 UI | `#60a5fa` 科技蓝 | 仪表盘、图表、数据流 |

### 关于"真实 UI"的特别说明

**Act 3（证据机器）**部分，所有出现的 UI 界面，必须使用 RealSourcing 的真实产品界面截图或录屏，而非后期制作的假 UI。这是这支视频与所有竞品最大的差异化——**我们展示的每一个数据，都是真实系统里可以查到的**。

如果产品 UI 尚未完全开发完毕，可以使用已完成的高保真 Figma 原型图，但绝对不能使用 Adobe After Effects 凭空制作的假数据界面。

### 音乐选曲方向

- **Act 1 (0-15s)**：低频心跳声（约 60 BPM），混合断断续续的键盘敲击声和 Slack 消息通知声。营造压迫感与焦虑感。
- **Act 2 (15-25s)**：在 15 秒处，用一个强烈的低音炮（Sub Bass Drop）打断所有声音，然后是 2 秒的完全静默。随后，一个充满力量感的合成器主旋律进入。
- **Act 3 (25-65s)**：节奏精准的电子乐（约 120 BPM），类似 Daft Punk 的 `Harder, Better, Faster, Stronger`，但更具科技感和现代感。
- **Act 4 (65-75s)**：音乐渐强至史诗感，然后在二维码出现的瞬间，收尾为一个单一的、悠长的合成器音符（类似 Hans Zimmer 在《星际穿越》中的手法）。

## v1.0 vs v2.0 核心差异对比

| 维度 | Gemini v1.0 | Manus v2.0 | 升级理由 |
| :--- | :--- | :--- | :--- |
| **核心叙事** | 解决"盲猜游戏"的痛点 | 消灭"增长税"这个隐形敌人 | "增长税"比"盲猜"更具体、更令人愤怒、更有传播性 |
| **价值定位** | 首席供应链增长伙伴 | **增长即服务 (Growth-as-a-Service)** | GaaS 概念更具 SaaS 时代感，更性感，更易被顶级卖家接受 |
| **证据呈现** | 后期制作的悬浮 HUD UI | **使用真实产品 UI 界面** | 真实 UI 的可信度远超特效，且能顺带展示产品成熟度 |
| **行动召唤** | 扫码预约 Webinar | **加入创始席位 (Founding Cohort)** | 创造稀缺性和归属感，顶级卖家对"抢先入场"的响应率远高于"注册试用" |
| **情绪弧线** | 痛点 → 解决方案 → 功能演示 | 痛点 → **认知颠覆** → 证据机器 → **邀请入场** | 增加了"认知颠覆"这个关键节点，让观众在接受解决方案前，先完成世界观的重建 |

| **Review & Delivery** | 2 轮修改、最终输出（4K + 1080p + 竖版 9:16 版本） | 3 天 |

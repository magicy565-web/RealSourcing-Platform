## 1. 项目背景与愿景 (Project Background & Vision)

**RealSourcing** 致力于打破跨国采购中的“信任黑盒”。通过 **AI + 实时音视频** 技术，将传统的、低效的、基于邮件和静态图片的采购模式，升级为**透明、实时、可追溯**的数字化选品体验。

**3.1 版本核心目标：**
- **信任闭环 (Trust Loop)**：通过声网云端录制与 AI 摘要，确保存证与追溯。
- **效率革命 (Efficiency)**：实时翻译与 AI Meeting Reel 消除语言障碍，缩短决策周期。
- **转化驱动 (Conversion)**：将直播与会议流量直接导向样品单与询价单。

## 2. 核心功能模块规划 (Core Modules)

### 2.1 实时音视频选品体系 (Real-time Sourcing System)
基于声网（Agora）底层能力，构建专业的 B2B 沟通环境。

| 功能点 | 详细描述 | 优先级 |
| :--- | :--- | :--- |
| **1:1 私密选品会** | 支持 RTC 高清视频、双向语音、低延迟交互。 | P0 |
| **互动白板 (Whiteboard)** | 支持在产品图片上进行实时标注、尺寸确认、细节勾画。 | P0 |
| **实时翻译 (STT+MT)** | 语音实时转文字并翻译（中英/中西/中日等），支持双语字幕显示。 | P0 |
| **AI 降噪与美颜** | 消除工厂背景噪音，提升主播专业形象。 | P1 |

### 2.2 信任与存证体系 (Trust & Evidence)
解决“所见即所得”与“承诺可追溯”的问题。

| 功能点 | 详细描述 | 优先级 |
| :--- | :--- | :--- |
| **云端录制 (Recording)** | 会议全程自动录制，文件加密存储于 S3，作为交易凭证。 | P0 |
| **AI 会议摘要 (Summary)** | 自动提取会议关键点（价格、MOQ、交期、定制要求）。 | P0 |
| **Meeting Reel 2.0** | 自动剪辑会议精彩片段，生成可分享的短视频产品报告。 | P1 |
| **工厂实地认证视频** | 强制关联工厂直播流与地理位置信息，防止虚假工厂。 | P2 |

### 2.3 交易转化闭环 (Transaction Loop)
缩短从“看到”到“下单”的路径。

| 功能点 | 详细描述 | 优先级 |
| :--- | :--- | :--- |
| **会议中一键申请样品** | 在视频通话侧边栏直接点击产品卡片，发起样品单。 | P0 |
| **直播间购物车 (Webinar Shop)** | 直播过程中实时推送热销品，支持买家即时询价。 | P0 |
| **样品单状态追踪** | 整合物流 API，实时反馈样品寄送进度。 | P1 |
| **智能询价助手** | AI 根据会议内容自动填充询价单草稿，买家确认即发。 | P1 |

## 3. 用户角色与核心流程 (User Personas & Workflows)

### 3.1 买家端 (Buyer Journey)
1.  **发现**：通过 Webinars 或 Factories 列表发现潜在供应商。
2.  **预约**：使用 BookMeeting 预约 1:1 选品会。
3.  **沟通**：进入 PrivateMeetingRoom，开启实时翻译与白板。
4.  **决策**：查看 AI 摘要，申请样品 (SampleOrder)。
5.  **回顾**：在 Reports 中查看 Meeting Reel。

### 3.2 工厂端 (Factory Journey)
1.  **展示**：通过 WebinarLive 发起新品发布直播。
2.  **接单**：在 FactoryDashboard 处理预约请求与样品单。
3.  **履约**：上传样品物流单号，维护产品库。

## 4. 技术架构要求 (Technical Requirements)

### 4.1 音视频架构 (Agora Integration)
- **RTC**: 使用 `agora-rtc-react` 实现多端同步。
- **RTM**: 负责信令、白板同步及实时翻译文本推送。
- **RESTful API**: 后端控制云端录制的开始与结束。

### 4.2 AI 引擎 (AI Engine)
- **STT**: 声网实时语音转文字。
- **LLM (GPT-4o)**: 负责会议摘要提取、翻译优化及 Meeting Reel 脚本生成。
- **Storage**: AWS S3 存储原始录制文件与剪辑后的视频。

---

## 5. 关键指标 (KPIs)

- **会议转化率**：会议结束后 24 小时内发起样品申请的比例（目标 > 15%）。
- **沟通效率**：单次选品会达成意向的平均时长（目标缩短 30%）。
- **用户留存**：买家查看 Meeting Reel 的频率与回访率。
